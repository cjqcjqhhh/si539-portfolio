About Genshin Impact

Genshin Impact, developed by Chinese developer miHoYo, is an action role-playing game that first published in 2020. The game features an anime-style open-world environment and an action-based battle system using elemental magic and character-switching. The game is free-to-play and is monetized through gacha game mechanics where players can obtain new characters, weapons, and other resources.

This simple Genshin Impact Wiki aims at demonstrating some basic characters, weapons and artifacts to beginners.
